# 👋 About Hohyun

안녕하세요, Hohyun입니다.  
백엔드부터 프론트엔드, 인프라, AI 시스템까지 전반을 다루는 **풀스택 개발자**입니다.  
이 블로그는 제가 경험하고 연구하는 기술들을 기록하고 공유하기 위해 운영하고 있습니다.

---

## 💻 기술 스택

### 🧠 AI & LLM
- LangChain, LangGraph  
- Llama, OpenAI API  
- RAG 시스템 설계 및 구현  
- 벡터DB 기반 검색 시스템 구축  

### ⚙️ Backend
- **Python** (FastAPI, Django)  
- **Java** (Spring Boot)  
- REST API / GraphQL  
- MSA 아키텍처 설계  
- 인증/인가, 캐싱, 비동기 처리  

### 🎨 Frontend
- **React**, **Next.js**  
- Zustand, Redux  
- TailwindCSS, Styled Components  

### 🗄 Database & Storage
- PostgreSQL  
- Redis  
- NeonDB  
- Upstash Redis  
- Supabase  

### 🛠 DevOps & Infra
- Docker  
- GitHub Actions  
- CI/CD 파이프라인 구축  
- Cloud 환경 운영
